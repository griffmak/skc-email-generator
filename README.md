# SKC Partnership Sales AI - Email Generator

AI-powered research and email generation for sports partnership sales.

## Quick Start (Local Development)

1. Install dependencies:
```bash
npm install
```

2. Run locally:
```bash
npm run dev
```

3. Open browser to `http://localhost:5173`

## Deploy to Vercel

1. Push this folder to GitHub
2. Go to vercel.com and sign in with GitHub
3. Click "Import Project"
4. Select this repository
5. Click "Deploy" (no configuration needed - Vercel auto-detects Vite)

## Features

- AI-powered company research using web search
- Generates 3 email variations (Conservative, Value-Focused, Bold)
- Multi-rightsholder management (portable across teams)
- Persistent storage with localStorage
- Copy-to-clipboard functionality

## Tech Stack

- React 18
- Vite (build tool)
- Tailwind CSS (styling)
- Lucide React (icons)
- Claude API (AI research & generation)
